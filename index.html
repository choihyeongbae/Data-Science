📐 코사인 유사도 시각화

두 벡터의 내적과 코사인 유사도를 실시간으로 확인해보세요

벡터 A (빨강)

벡터 B (파랑)

내적 (Dot Product)

0
A · B = Ax·Bx + Ay·By

벡터 A 크기

0
||A|| = √(Ax² + Ay²)

벡터 B 크기

0
||B|| = √(Bx² + By²)

코사인 유사도

0
cos θ = (A·B) / (||A||·||B||)

각도 (θ)

0°
θ = arccos(cos θ)

💡 코사인 유사도란?

코사인 유사도는 두 벡터 사이의 각도를 이용하여 유사도를 측정하는 방법입니다. 값의 범위는 -1에서 1까지이며, 1에 가까울수록 두 벡터가 같은 방향을 가리키고, 0에 가까우면 직각, -1에 가까우면 반대 방향을 의미합니다. 추천 시스템, 문서 유사도 분석 등에서 널리 활용됩니다.
