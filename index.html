01

데이터란 무엇인가?

데이터(Data)는 현실 세계의 관찰, 측정, 수집을 통해 얻어진 사실(fact)이나 값(value)의 집합입니다. 라틴어 "datum"(주어진 것)에서 유래했으며, 그 자체로는 의미가 없는 원시적 상태입니다.

💡 데이터 → 정보 → 지식 → 지혜

데이터: "37.5" — 그 자체로는 의미 없는 숫자
정보: "환자의 체온이 37.5°C" — 맥락이 부여된 데이터
지식: "37.5°C 이상이면 발열로 판단" — 정보에 해석이 더해진 것
지혜: "발열 환자에게 해열제를 투여하고, 추이를 관찰해야 한다" — 지식의 활용

데이터과학에서는 이 원시 데이터를 수집·정제·분석하여 의미 있는 정보와 인사이트를 추출합니다. 머신러닝 모델도 결국 데이터에서 패턴을 학습하는 것이므로, 좋은 데이터 없이는 좋은 AI도 없습니다.

🗣️ "Garbage In, Garbage Out"

데이터 과학계의 유명한 격언입니다. 아무리 좋은 알고리즘이라도 품질 낮은 데이터를 넣으면 쓸모없는 결과가 나옵니다. 그래서 데이터를 제대로 이해하는 것이 먼저입니다.

02

데이터의 유형

데이터는 구조화 정도에 따라 크게 세 가지로 분류합니다.

📊

정형 데이터 Structured

행과 열로 구성된 표(Table) 형태. 엑셀, CSV, 데이터베이스에 저장. 프로그래밍으로 처리하기 가장 쉬운 형태입니다.

📝

반정형 데이터 Semi-structured

JSON, XML, HTML 등. 완전한 표는 아니지만 태그나 키-값 쌍으로 일정한 구조를 가집니다.

🖼️

비정형 데이터 Unstructured

이미지, 영상, 음성, 텍스트 문서 등. 전체 데이터의 약 80%를 차지하며, AI 기술로 분석합니다.

📝 반정형 데이터란?

반정형 데이터는 표처럼 깔끔하진 않지만, 나름의 규칙을 가진 데이터입니다. 예를 들어 JSON 데이터는 {"이름": "김민지", "나이": 17} 처럼 키-값 쌍으로 구성되어 있어 프로그래밍으로 구조를 파악할 수 있습니다. 우리가 사용하는 공공데이터 API 대부분이 JSON 형태로 제공되기 때문에, 이 유형을 이해하는 것이 중요합니다.

03

데이터의 속성 (변수의 유형)

데이터를 분석할 때, 각 열(변수/feature)이 어떤 속성인지 파악하는 것이 매우 중요합니다. 속성에 따라 사용할 수 있는 분석 방법과 시각화 방식이 달라집니다.

대분류소분류설명예시
수치형연속형무한히 세분화할 수 있는 측정값키, 몸무게, 온도, 시간
이산형셀 수 있는 정수 값학생 수, 판매 개수, 클릭 횟수
범주형명목형순서 없는 카테고리혈액형, 성별, 과목명
순서형순서가 있는 카테고리학점(A>B>C), 만족도(상/중/하)

04

데이터 전처리

데이터 전처리(Data Preprocessing)란 수집된 원시 데이터를 분석이나 모델 학습에 적합한 형태로 변환하는 과정입니다. 실제 데이터 과학 프로젝트에서 전체 작업 시간의 60~80%가 전처리에 소요됩니다.

전처리가 필요한 이유

🕳️

결측치 (Missing Values)

비어있는 값. 설문 미응답, 센서 오류 등으로 발생.

👾

이상치 (Outliers)

다른 데이터와 극단적으로 다른 값. 입력 오류이거나 특이 사례.

🔄

중복 데이터

같은 데이터가 여러 번 입력됨. 분석 결과를 왜곡시킴.

📏

스케일 차이

키(170cm)와 몸무게(65kg)처럼 단위와 범위가 다른 변수들.

🏷️

불일치 형식

"서울", "서울시", "Seoul" 등 같은 값이 다른 형태로 입력된 경우.

🔤

잘못된 타입

숫자가 문자열로 저장된 경우 ("1,500원" → 1500).

09

이 수업의 학습 로드맵

이번 수업은 Google Colab + Python 환경에서 진행됩니다. 아래 순서로 데이터 분석의 전체 파이프라인을 경험하게 됩니다.

STEP 01

🐍 Python 기초

변수, 자료형, 선택 구조 등 프로그래밍의 기본 문법을 학습합니다.

STEP 02

📦 자료구조: 리스트, 딕셔너리

데이터를 담는 그릇. 리스트(배열), 딕셔너리(키-값)를 자유롭게 다룹니다.

STEP 03

🔢 NumPy

수치 연산의 핵심 라이브러리. 배열 연산, 통계 함수, 벡터 연산을 학습합니다.

STEP 04

🐼 Pandas

데이터 분석의 핵심! DataFrame으로 데이터를 읽고, 정제하고, 탐색합니다.

STEP 05

📊 Matplotlib

데이터를 시각화합니다. 막대, 선, 산점도, 히스토그램 등 다양한 차트를 그립니다.

STEP 06

🔍 EDA(탐색적 데이터 분석)

공공데이터를 가져와서 전처리하고, 시각화하며 패턴과 인사이트를 발견합니다.
