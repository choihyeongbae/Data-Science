01

데이터란 무엇인가?

데이터(Data)는 현실 세계의 관찰, 측정, 수집을 통해 얻어진 사실(fact)이나 값(value)의 집합입니다. 라틴어 "datum"(주어진 것)에서 유래했으며, 그 자체로는 의미가 없는 원시적 상태입니다.

💡 데이터 → 정보 → 지식 → 지혜

데이터: "37.5" — 그 자체로는 의미 없는 숫자
정보: "환자의 체온이 37.5°C" — 맥락이 부여된 데이터
지식: "37.5°C 이상이면 발열로 판단" — 정보에 해석이 더해진 것
지혜: "발열 환자에게 해열제를 투여하고, 추이를 관찰해야 한다" — 지식의 활용

데이터과학에서는 이 원시 데이터를 수집하고, 정제하고, 분석하여 의미 있는 정보와 인사이트를 추출합니다. 머신러닝 모델도 결국 데이터에서 패턴을 학습하는 것이므로, 좋은 데이터 없이는 좋은 AI도 없습니다.

🗣️ "Garbage In, Garbage Out"

데이터 과학계의 유명한 격언입니다. 아무리 좋은 알고리즘이라도 질 낮은 데이터를 넣으면 쓸모없는 결과가 나옵니다. 이 수업에서 데이터를 먼저 배우는 이유가 바로 여기에 있습니다.

데이터의 유형

데이터는 구조화 정도에 따라 크게 세 가지로 분류합니다.

📊

정형 데이터 Structured

행과 열로 구성된 표(Table) 형태. 엑셀, CSV, 데이터베이스에 저장. 프로그래밍으로 처리하기 가장 쉬운 형태입니다.

📝

반정형 데이터 Semi-structured

JSON, XML, HTML 등. 완전한 표는 아니지만 태그나 키-값 쌍으로 일정한 구조를 가집니다.

🖼️

비정형 데이터 Unstructured

이미지, 영상, 음성, 텍스트 문서 등. 전체 데이터의 약 80%를 차지하며, AI 기술로 분석합니다.

04

데이터 전처리

데이터 전처리(Data Preprocessing)란 수집된 원시 데이터를 분석이나 모델 학습에 적합한 형태로 변환하는 과정입니다. 실제 데이터 과학 프로젝트에서 전체 작업 시간의 60~80%가 전처리에 소요됩니다.

전처리가 필요한 이유

🕳️

결측치 (Missing Values)

비어있는 값. 설문 미응답, 센서 오류 등으로 발생. NaN, null, 빈칸 등으로 나타남.

👾

이상치 (Outliers)

다른 데이터와 극단적으로 다른 값. 입력 오류이거나 실제 특이 사례일 수 있음.

🔄

중복 데이터

같은 데이터가 여러 번 입력됨. 분석 결과를 왜곡시킬 수 있음.

📏

스케일 차이

키(170cm)와 몸무게(65kg)처럼 단위와 범위가 다른 변수들. 정규화/표준화 필요.

🏷️

불일치 형식

"서울", "서울시", "Seoul" 등 같은 값이 다른 형태로 입력된 경우.

🔤

잘못된 타입

숫자가 문자열로 저장된 경우 ("1,500원" → 1500). 타입 변환이 필요.

06

측정 척도 (Scale of Measurement)

데이터를 더 깊이 이해하려면 측정 척도를 알아야 합니다. 같은 숫자라도 척도에 따라 의미와 가능한 연산이 완전히 달라집니다.

척도특징예시가능한 연산
명목 척도 분류만 가능, 순서 ✕ 혈액형(A,B,O,AB), 성별, 지역 빈도, 최빈값
순서 척도 순서 O, 간격 ✕ 학점(A>B>C), 만족도(1~5) 빈도, 최빈값, 중앙값
등간 척도 순서 O, 간격 동일, 절대 0 ✕ 온도(°C), IQ 점수, 연도 +평균, 표준편차
비율 척도 순서 O, 간격 동일, 절대 0 O 키, 몸무게, 속도, 수입 +비율 비교 ("2배")

⚠️ 왜 척도가 중요할까?

명목 척도인 혈액형을 A=1, B=2, O=3, AB=4로 코딩했다고 해서 "AB형은 A형의 4배"라고 말할 수 없습니다. 머신러닝에서도 명목형 변수를 그냥 숫자로 넣으면 모델이 크기 관계를 잘못 학습합니다. 그래서 원-핫 인코딩(One-Hot Encoding) 같은 기법이 필요합니다.

🐍 원-핫 인코딩 예시

import pandas as pd df = pd.DataFrame({'혈액형': ['A','B','O','AB','A']}) # 원-핫 인코딩 encoded = pd.get_dummies(df['혈액형'], prefix='혈액형') print(encoded) # 결과: # 혈액형_A 혈액형_AB 혈액형_B 혈액형_O # 0 True False False False # 1 False False True False # ...

📦 자료구조: 리스트, 딕셔너리

STEP 03

🔢 NumPy

수치 연산의 핵심 라이브러리. 배열 연산, 통계 함수, 벡터 연산을 학습합니다.

STEP 04

🐼 Pandas

데이터 분석의 핵심! DataFrame으로 데이터를 읽고, 정제하고, 탐색합니다. CSV/Excel 처리의 핵심.

STEP 05

📊 Matplotlib

데이터를 시각화합니다. 막대, 선, 산점도, 히스토그램 등 다양한 차트를 그립니다.
